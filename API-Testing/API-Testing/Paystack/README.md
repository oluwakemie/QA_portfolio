# Paystack API Testing Project

## Overview
This project demonstrates API testing of Paystack payment services using Postman.
The scope includes transaction initialization, verification, authorization charges,
webhook validation, and negative testing scenarios.

## Tools Used
- Postman (API Testing & Automation)
- JavaScript (Postman test scripts)
- GitHub (Version Control & Documentation)

## Test Coverage
- Positive & Negative API scenarios
- Authentication & authorization validation
- Response schema validation
- Environment variable chaining
- Webhook security verification
- Performance & reliability checks

## How to Run
1. Import the Postman collection
2. Set environment variables:
   - `PAYSTACK_SECRET_KEY`
   - `base_url`
3. Run collection using Postman Runner

## Expected Outcome
- Validate API reliability
- Ensure secure payment transactions
- Detect errors before production release
